### 排序分类

排序可以简单分为**插入排序**、**交换排序**、**选择排序**、**归并排序**、**基数排序**

1. 插入排序：每次将一个元素插入到已排序部分的合适位置（(折半)插入、希尔排序）

2. 交换排序：每次交换相邻元素使其有序（冒泡排序、快速排序）

3. 选择排序：每次选择最小（大）元素放到已排序部分（选择排序、堆排序）

其中，只有基数排序是非比较排序

### 稳定性

对于稳定排序，一般通过不取等来实现稳定；取等后则不稳定。也就是说，稳定排序可以手动变得不稳定

对于不稳定排序，则没有办法使其变得稳定

1. 不稳定排序：希尔排序、快速排序、选择排序、堆排序

2. 稳定排序：(折半)插入排序、冒泡排序、归并排序、基数排序
